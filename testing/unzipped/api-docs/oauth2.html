oauth2 package

public object OAuth2ClientEndpointConfiguration

OAuth2ClientEndpointConfiguration represents options to be used for OAuth2 client invocation

Field Name Data Type Description Default Value
accessToken string Access token for OAuth2Endpoint
baseUrl string The base url of the api
clientId string The client id
clientSecret string The client secret
refreshToken string The refresh token
refreshTokenEP string The base endpoint url to use refresh token
refreshTokenPath string The path of refresh token url
useUriParams boolean Specifies whehter use uri parameters. The default value is false. false
setCredentialsInHeader boolean accessToken: Access token for OAuth2Endpoint false
clientConfig ClientEndpointConfiguration ClientEndpointConfiguration to use for HTTP client invocation
